## 1) Local-First Default (Offline Works)

- The Software is designed to function fully offline for normal cabinet operation and gameplay.
- No account is required for local play.
- Declining consent does not block basic cabinet use.
- Online Features are optional enhancements and may be disabled.

## 2) Consent & Opt-In Model (Granular + Revocable)

Certain Online Features require explicit opt-in consent (for example via an on-screen consent overlay). Consent is granular and revocable at any time from settings (or by disabling Online Features).

**Requires consent (examples):**
- Public or shared leaderboards
- Score syncing across cabinets
- Display name / nickname visibility outside the cabinet
- Use of a cabinet identifier for syncing/leaderboards/integrity
- Optional communications (email/SMS) only if enabled and provided

**Does NOT require consent:**
- Local gameplay
- Local scores / local profiles
- Basic cabinet operation without internet

## 3) Data We Collect (Specific + Minimal)

**A) Local-only by default (no sharing unless you opt in):**
- Local profiles, local scores, settings, and configuration stored on the cabinet device

**B) If you opt into Online Features, the Software may transmit/store:**
- Display name / nickname (as entered)
- Game titles played
- Scores/achievements and timestamps
- Cabinet ID / device identifier (for syncing, moderation, integrity, and abuse prevention)
- Basic configuration metadata (non-personal, e.g., feature toggles, software version)

**C) Data we do not intentionally collect:**
- Government IDs
- Biometric identifiers
- Payment card data (unless you add a separate payment system outside this Software)
- Real names unless a user types one in
- Permanent voice recordings unless explicitly enabled (see Voice & Audio)

## 4) Voice & Audio (Microphone-Only)

Arcade Assistant may offer optional voice features. If voice features are enabled, the cabinet microphone receives audio input only to provide the requested functionality (for example, voice commands or spoken assistance). Voice features are optional and can be disabled.

- Arcade Assistant does not use voice data to identify individuals.
- Arcade Assistant does not sell voice data.
- Unless you explicitly enable a setting that saves transcripts/logs, voice audio and any resulting transcripts are intended to be processed transiently and not stored long-term by Arcade Assistant.
- If Online Features are enabled, voice processing may rely on third-party services (see Third-Party Services).
- **Public-space reminder:** Cabinets may be used in public venues; users should avoid speaking sensitive information if they do not want it processed for voice functionality.
