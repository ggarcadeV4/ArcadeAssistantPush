## ⚠️ Important Gotchas & Lessons Learned

### PanelShell Component
- **Before this session**: PanelShell didn't support custom header buttons
- **Now**: Supports `icon`, `subtitle`, and `headerActions` props
- **Why it matters**: Other panels can now add header actions (chat, settings, etc.)

### Chat Sidebar Pattern
- **Working pattern**: Conditional render with `{chatOpen && <div>...</div>}`
- **Failed pattern**: CSS animation with `right: -450px` transition
- **Use**: Fixed positioning (`position: fixed; right: 0; top: 0`) with conditional rendering

### Avatar Integration
- **Image avatars**: Use `<img src="/avatar.jpeg" className="voice-avatar-img" />`
- **Emoji fallbacks**: Use `<div className="voice-avatar">{emoji}</div>`
- **Conditional logic**: Check `avatar.startsWith('/')` to determine which to render

### Uniform Component Styling
- **Don't**: Use dynamic classes like `.active` based on data state
- **Do**: Apply consistent styling; let content indicate state
- **Why**: Prevents visual imbalance and provides cleaner UX

### Proper Chat Message Sending Pattern
```javascript
const sendMessage = async () => {
  if (!input.trim() || isLoading) return

  const userMessage = { role: 'user', text: input }
  setMessages(prev => [...prev, userMessage])
  setInput('')
  setIsLoading(true)

  try {
    const response = await fetch('/api/ai/chat', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-scope': 'state',
        'x-device-id': localStorage.getItem('deviceId') || crypto.randomUUID()
      },
      body: JSON.stringify({
        messages: [...messages, userMessage],
        // Add personality context from selected voice profile
        systemPrompt: `You are ${selectedProfile.name}...`
      })
    })

    const data = await response.json()
    setMessages(prev => [...prev, { role: 'assistant', text: data.message }])
  } catch (error) {
    console.error('Chat error:', error)
    // Show error to user
  } finally {
    setIsLoading(false)
  }
}
```

### Microphone Permission Handling (Echo Agent)
```javascript
const requestMicPermission = async () => {
  try {
    const stream = await navigator.mediaDevices.getUserMedia({ audio: true })
    setHasMicPermission(true)
    return stream
  } catch (error) {
    if (error.name === 'NotAllowedError') {
      setError('Microphone permission denied. Please enable in browser settings.')
    } else if (error.name === 'NotFoundError') {
      setError('No microphone detected. Please connect a microphone.')
    }
    setHasMicPermission(false)
    return null
  }
}
```
